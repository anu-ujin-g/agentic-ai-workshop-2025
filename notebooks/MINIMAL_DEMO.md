## All Prompts (Complete List)

1. `Search: ` - Main input
2. `Where: ` - Only if no location in search
3. `Feedback? (y/n): ` - Optional feedback
4. `Which? (1/2): ` - If giving feedback
5. `Liked? (y/n): ` - If giving feedback
6. `Vibe (1-5): ` - If liked
7. `Visited? (y/n): ` - If giving feedback
8. `Notes: ` - If giving feedback

**That's all!**
